Elements
These are the custom elements created by using the Themeify Library. Users can choose to use these elements to easily create a Neon Themed website.
Fonts
You can use the funtion "changeFont" in provided in every element generators to change fonts. Here are the available fonts for theme "Neon":
Megrim (default) Monoton (Title) Comfortaa
Title
Regular Size Title: DEFAULT
Large Size Title: themeify('darkPurple','skyBlue')
Small Size Title: themeify('blue','cyan')
Animations: Title Elements
The animations for Neon Theme are focused on the Title elements. These animations are created using DOM manipulation and JS Events. They can be applied to elements created by NeonTitleGenerator. The animation functions can also be applied to subtitles.
